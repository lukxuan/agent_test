# Auto Parking System - Product Requirements Document (PRD)

## 1. 产品概述

### 1.1 产品名称
Auto Parking System（自动泊车系统）

### 1.2 产品定位
面向自动驾驶/高级辅助驾驶（ADAS）领域的自动泊车算法框架（纯算法库，不含仿真环境），提供车位检测、路径规划、运动控制的全链路算法能力。

### 1.3 目标用户
自动驾驶算法工程师、ADAS 系统开发者。

## 2. 功能需求

### 2.1 泊车场景

| 场景 | 描述 | 车位方向 |
|------|------|----------|
| 平行泊车 (Parallel) | 路边侧方停车 | 车位与行驶方向平行 |
| 垂直泊车 (Perpendicular) | 停车场倒车入库 | 车位与行驶方向垂直 |
| 斜向泊车 (Diagonal) | 倾斜车位停车 | 车位与行驶方向成 30°-60° 夹角 |

### 2.2 功能模块

#### FR-01: 车位检测（Perception）
- **FR-01-1**: 支持基于超声波传感器数据的停车位检测
- **FR-01-2**: 输出车位类型、位置（四角坐标）、尺寸（宽×长）、朝向角度
- **FR-01-3**: 支持对三种泊车场景的车位识别

#### FR-02: 路径规划（Path Planning）
- **FR-02-1**: 根据车辆当前位置和目标车位，生成无碰撞泊车路径
- **FR-02-2**: 平行泊车采用"两段式圆弧 + 直线"轨迹
- **FR-02-3**: 垂直泊车采用"前进调整 + 倒车圆弧"轨迹
- **FR-02-4**: 斜向泊车采用"角度自适应弧线"轨迹
- **FR-02-5**: 路径点包含位置、朝向、曲率、速度信息
- **FR-02-6**: 支持碰撞检测（车辆包围盒 vs 障碍物）

#### FR-03: 运动控制（Motion Control）
- **FR-03-1**: 基于纯追踪（Pure Pursuit）算法的横向控制
- **FR-03-2**: 基于 PID 的纵向速度控制
- **FR-03-3**: 输出转向角和速度控制指令
- **FR-03-4**: 满足车辆运动学约束（最大转向角、最大速度）

#### FR-04: 泊车状态管理（Parking Scenario）
- **FR-04-1**: 状态机管理：IDLE → SEARCHING → PLANNING → EXECUTING → COMPLETED/FAILED
- **FR-04-2**: 根据车位类型自动选择对应规划策略
- **FR-04-3**: 支持泊车过程中异常检测与回退机制

#### FR-05: 顶层接口（Parking System）
- **FR-05-1**: 提供 `startParking()`、`update()`、`getState()` 等简洁 API
- **FR-05-2**: 模块解耦，各模块可独立使用

## 3. 输入输出规范

### 3.1 输入

| 输入项 | 类型 | 说明 |
|--------|------|------|
| 传感器数据 | SensorData | 超声波测距值序列 |
| 车辆位姿 | Pose2D | 当前位置 (x,y) + 航向角 yaw |
| 车辆参数 | VehicleParams | 轴距、轮距、车长、车宽、最小转弯半径等 |
| 目标车位 | ParkingSlot | 类型、四角坐标、尺寸、角度 |

### 3.2 输出

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 控制指令 | ControlCommand | 目标转向角 (rad)、目标速度 (m/s) |
| 泊车状态 | ParkingState | 当前状态机状态 |
| 规划路径 | vector<PathPoint> | 路径点序列 |

## 4. 性能指标

| 指标 | 要求 |
|------|------|
| 路径规划耗时 | < 50ms（单次规划） |
| 运动控制计算 | < 10ms（单次更新） |
| 泊车最终位姿精度 | 位置误差 < 10cm，角度误差 < 3° |
| 最终位姿居中度 | 车辆相对车位中心偏移 < 15cm |

## 5. 约束条件

### 5.1 车辆约束

| 参数 | 范围 | 默认值 |
|------|------|--------|
| 轴距 (wheelbase) | 2.5m - 3.5m | 2.8m |
| 车长 (length) | 4.0m - 5.5m | 4.6m |
| 车宽 (width) | 1.7m - 2.2m | 1.85m |
| 最大转向角 | 30° - 45° | 35° |
| 最大倒车速度 | 1.0 - 3.0 m/s | 1.5 m/s |

### 5.2 车位约束

| 场景 | 最小车位宽度 | 最小车位长度 |
|------|-------------|-------------|
| 平行泊车 | 车宽 + 0.6m | 车长 + 0.8m |
| 垂直泊车 | 车宽 + 0.4m | 车长 + 0.5m |
| 斜向泊车 | 车宽 + 0.3m | 车长 + 0.5m |

### 5.3 环境约束
- 工作温度：-20°C ~ 60°C
- 坡度：≤ 5%
- 假设车位两侧有静止障碍物（已停车辆）
- 假设路面平整，摩擦系数充足

## 6. 非功能需求

- **NFR-01**: C++17 标准，跨平台（Windows/Linux）
- **NFR-02**: 模块化设计，各模块通过接口解耦
- **NFR-03**: 无外部第三方依赖（仅 GoogleTest 用于测试）
- **NFR-04**: 代码覆盖率达到核心模块 ≥ 80%
